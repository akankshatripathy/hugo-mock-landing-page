Mobile Application 🛡️

Crowdsourcing iOS application where users walk with phone to track the sidewalks where potholes could occur

Computer Vision 🚀

Use a classification algorithm to detect whether or not there are potholes on roads

React Desktop App 📈

Use Google Maps API to pull the location of the pothole from the backend and display on the map with the desktop app so municipalities can fix
